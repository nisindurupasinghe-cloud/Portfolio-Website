# Portfolio-Website
A clean, responsive, and modern portfolio website built entirely with HTML, CSS, and JavaScript. Designed to showcase my projects, skills, and background as a Computer Engineering undergraduate and tech enthusiast. 

🚀 Features

• Fully Responsive – Works on mobile, tablet, and desktop

• Modern UI/UX – Clean design with smooth animations

• Interactive Elements – Dynamic navigation and hover effects 

• SEO-Friendly – Semantic HTML and structured content

• Fast & Lightweight – No external libraries, pure vanilla JS and CSS

• Accessible – Follows web accessibility best practices


🛠️ Tech Stack

• HTML5 – Semantic structure

• CSS3 – Custom styling, Grid, Flexbox, animations

• JavaScript (ES6+) – DOM manipulation and interactivity

• Git & GitHub – Version control and deployment


📁 Project Structure

• index.html – Main HTML file

• style.css – Main stylesheet

• script.js – JavaScript for interactivity

• README.md – Project documentation


🧩 Sections Included

• Home – Introduction and call-to-action

• About – Personal bio and background

• Education – Academic journey and certifications

• Projects – Showcase of technical projects

• Skills – Technical and soft skills

• Contact – Contact form and social links


🧠 What I Learned

• Building a complete frontend project from scratch

• Implementing responsive layouts with CSS Grid and Flexbox

• Writing clean, maintainable, and reusable CSS

• Enhancing UX with JavaScript-driven interactions

• Structuring semantic HTML for accessibility and SEO

• Managing project workflow with Git and GitHub
